Q: Your tests don't work! Nothing builds!
A: Set your target platform as x64 for building and testing. 
Building is right next to the 'Start' button, testing is Test->Test Settings->Default Processor Architecture->X64

Q: I can't connect to the robot!
A: Check if the IP of the robot matches the IP in the code.

Q: I get an exception when I try to 3D scan!
A: Have you installed the Kinect Driver? Try testing other Kinect applications to see if they work.

Q: Still get an exception regarding the Kinect.
A: The driver might be newer than the libraries. Try replacing the Kinect .dll's
